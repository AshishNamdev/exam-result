# Govt. Sandipani Model HS School Ashok Nagar - Exam Results Portal

This repository hosts the **Class 09 & 11 Annual Exam Results (2025‑26)** page for Govt. Sandipani Model HS School, Ashok Nagar.  
It is published using **GitHub Pages** and allows students to check their exam results online.

---

## 🚀 Features
- Simple and clean interface
- Roll number validation:
  - Only accepts **13‑digit** (Class 09) or **14‑digit** (Class 11) roll numbers
  - Rejects empty or alphanumeric input
- Redirects to the official exam results portal
- Error messages displayed clearly for invalid input
- Opens results in a **new browser tab**
- Includes school logo and contact details for support

---

## 📖 How to Use
1. Visit the hosted site:  
   `https://<your-username>.github.io/exam-results/`
2. Enter your roll number (13 or 14 digits only).
3. Click **"परिणाम देखें"** (View Result).
4. You will be redirected to the official exam results page.

---

## 🛠️ Hosting on GitHub Pages
1. Create a new repository (e.g., `exam-results`).
2. Upload the following files:
   - `index.html`
   - `logo.png` (school logo)
3. Commit changes.
4. Go to **Settings → Pages**.
5. Under **Source**, select:
   - Branch: `main`
   - Folder: `/root`
6. Save. Your site will be live at:  
   `https://<your-username>.github.io/exam-results/`
